# frontend-template
 Develop your frontend projects with webpack, es6+, scss and more.

## What do I get?
- webpack
- babel
- eslint
- stylelint
- scss
- es6+
- autoprefixer

## Getting started
- Use this template
- `npm i`
- Now you are ready to go.

## npm scripts
- `npm run watch` - Start webpack in watch mode
- `npm run build` - Build assets once
- `npm run lint` - Lint via eslint & stylelint
